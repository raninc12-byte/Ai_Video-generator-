## How to Use

1. **Launch**: Double-click `run.bat`
2. **Describe**: Type what you want (e.g., "Beautiful sunset animation")
3. **Adjust**: Use slider for frame count (3-30 frames)
4. **Generate**: Click "🎥 GENERATE VIDEO"
5. **Done**: Video appears in Downloads/AI Videos

## Troubleshooting

**GUI doesn't open?**
→ Try: `python video_gen.py` in command prompt

**Dependencies issue?**
→ Run: `python -m pip install Pillow imageio imageio-ffmpeg`

**Where are my videos?**
→ Check: `C:\Users\[YOU]\Downloads\AI Videos\`
